# 🌟 Project Overview

RevHire is a **modern job recruitment platform** built using **Spring Boot** that connects **Employers and Job Seekers**.

The platform enables employers to post jobs and manage applicants while job seekers can apply for jobs and manage their professional profiles.

---

# 🎯 Objectives

✔ Simplify recruitment process  
✔ Provide centralized hiring platform  
✔ Allow job seekers to manage resumes and profiles  
✔ Enable employers to track and manage applicants  

---

# ✨ Key Features

## 👤 User Management
- Secure login and registration
- Password reset functionality
- Role-based authentication

## 💼 Job Management
- Employers can create job postings
- Edit and delete jobs
- View applicants

## 📄 Resume Upload
- Upload and manage resumes
- Store candidate profiles

## 🔔 Notification System
- Application status updates
- Job alerts

## 📊 Dashboard
- Employer dashboard
- Job seeker dashboard
- Application tracking

# 🏗 System Architecture

```
Client Browser
      │
      ▼
Frontend (HTML + CSS + Thymeleaf)
      │
      ▼
Spring Boot Controllers
      │
      ▼
Service Layer (Business Logic)
      │
      ▼
Repository Layer (Spring Data JPA)
      │
      ▼
Database (H2 / SQL)
```

# 🚀 Getting Started

## Clone Repository

```
git clone https://github.com/riyazshaik52/RevHire.git
```

## Navigate to Project

```
cd RevHire
```

## Run Application

```
mvn spring-boot:run
```

## Open Browser

```
http://localhost:8080
```
